# Exile Models

This is the NestJs back end for my model railway tools projects

## References
- [WiThrottle Protocol](https://jmri.org/help/en/package/jmri/jmrit/withrottle/Protocol.shtml)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
